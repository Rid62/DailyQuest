# 🎮 DailyQuest - تطبيق تحديات يومية مع نظام ذكاء اصطناعي

> تطبيق ويب تفاعلي لتتبع التحديات اليومية والعادات الصحية مع نظام نقاط وتصعيد مستويات

## ✨ الميزات الرئيسية

### 🎯 نظام التحديات الشامل
- ✅ **تحديات يومية ذكية** - تُولّد باستخدام OpenAI GPT-3.5
- ✅ **5 فئات تحديات**: إبداعية، تعليمية، اجتماعية، صحية، كسر الروتين
- ✅ **نظام نقاط متقدم** - اكسب نقاط وارتفع في المستويات
- ✅ **تتبع الإنجازات** - مشاهدة شارية الإنجازات والأوسمة

### 🛡️ نظام مكافحة الغش (جديد)
- 📝 **إثبات الإنجاز**: يجب كتابة 50+ حرف لوصف الإنجاز قبل الحصول على النقاط
- 🎯 **Proof Types**: كل فئة تحديات لها نوع إثبات محدد
- 🔒 **تحقق فوري**: عداد أحرف وتحقق من الحد الأدنى
- 💾 **حفظ الإثباتات**: تخزين دائم لجميع الإثباتات المقدمة

### 🤖 شخصيات محسّنة بالذكاء الاصطناعي
- 🎯 **ملف شخصي ذكي**: يتذكر أهدافك ومعلوماتك الشخصية
- 💭 **تحديات مخصصة**: تُولّد خصيصاً لك بناءً على ملفك الشخصي
- ✍️ **تحليلات ذكية**: توصيات شخصية لتحسين عاداتك

### 🎪 واجهة مستخدم احترافية
- 🌈 **تصميم جميل**: ألوان متدرجة وتأثيرات سلسة
- 📱 **responsive**: يعمل على جميع الأجهزة
- ⚡ **تفاعلي**: بدون إعادة تحميل صفحة (Reactive)
- 🎉 **احتفالات فورية**: Modal عند الترقية لمستوى أعلى

### 🏆 نظام الإحصائيات
- 📊 **رسوم بيانية**: تتبع التقدم في آخر 30 يوم
- 🔥 **شريط الإنجازات**: تابع انقطاعك المتتالي
- 🏅 **أوسمة وشهادات**: اكسب أوسمة بإنجازات معينة
- 🥇 **لوحة الصدارة**: تنافس مع مستخدمين آخرين

### 🛒 نظام المتجر (Shop)
- 🧊 **تجميد الشريط**: احم شريطك من الانقطاع (100 نقطة)
- 🎁 **مكافآت مستقبلية**: عناصر جديدة قريباً
- 💰 **نقاط واقعية**: اكسب نقاط من التحديات واستخدمها

### 🌈 ألوان ديناميكية (Hot Streaks)
- 🔵 **أزرق**: شريط عادي (0-2 أيام)
- 🔥 **برتقالي ناري**: شريط قوي (3-6 أيام)
- 💜 **بنفسجي غالاكسي**: شريط مذهل (7+ أيام)

## 🚀 البدء السريع

### المتطلبات:
- Python 3.10+
- pip
- مفتاح OpenAI API (اختياري للتحديات الذكية)

### التثبيت:

```bash
# 1. استنساخ المشروع
git clone https://github.com/Rid62/DailyQuest.git
cd DailyQuest

# 2. إنشاء بيئة افتراضية
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 3. تثبيت المتطلبات
pip install -r requirements.txt

# 4. إعداد متغيرات البيئة
echo "OPENAI_API_KEY=your-api-key-here" > .env
echo "SECRET_KEY=your-secret-key-here" >> .env

# 5. تشغيل التطبيق
python -m flask --app app run
```

**الرابط:** http://localhost:5000

## 📖 الاستخدام

### 1️⃣ التسجيل والدخول
```
- اضغط "Register" لإنشاء حساب جديد
- أدخل اسم مستخدم وبريد وكلمة مرور
- ادخل لصفحة التخصيص (Onboarding)
```

### 2️⃣ الملف الشخصي
```
أخبرنا عن:
- عمرك
- حالتك الوظيفية
- أهدافك
- التحديات التي تواجهها
```

### 3️⃣ التحديات اليومية
```
- اذهب إلى Dashboard
- شاهد 5 تحديات يومية
- اضغط "Mark as Complete"
- اكتب إثبات الإنجاز (50+ حرف)
- احصل على النقاط والمستوى يرتفع
```

### 4️⃣ تتبع التقدم
```
- شاهد الرسوم البيانية
- اعرض أوسمتك
- تنافس على لوحة الصدارة
- قم بشراء عناصر من المتجر
```

## 🔌 API Endpoints

### المصادقة:
- `POST /register` - إنشاء حساب جديد
- `POST /login` - تسجيل الدخول
- `GET /logout` - تسجيل الخروج

### التحديات:
- `GET /dashboard` - لوحة المعلومات الرئيسية
- `POST /api/complete-challenge/<id>` - إكمال تحدي مع الإثبات
- `GET /api/daily-challenge` - التحدي اليومي الرئيسي

### الإحصائيات:
- `GET /achievements` - الأوسمة والإنجازات
- `GET /leaderboard` - لوحة الصدارة
- `GET /api/user-stats` - إحصائياتك

### المتجر:
- `GET /shop` - متجر العناصر
- `POST /api/buy-item` - شراء عنصر
